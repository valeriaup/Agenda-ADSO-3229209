## Aprendizajes Clave

Durante esta semana, reforzamos la importancia de los métodos funcionales de JavaScript para escribir código más legible y eficiente. Además, se fortaleció el flujo de trabajo mediante la terminal, integrando Git como herramienta fundamental para el seguimiento de evidencias y control de versiones.

## Desafíos y Soluciones

### 1. Comandos de Git en la Terminal
* Dificultad: Curva de aprendizaje inicial con comandos en el CMD.
* Solución: Consulta constante de la documentación y práctica repetitiva de comandos como git status, add, commit y push.

### 2. Selección de Métodos de Arreglos
* Dificultad: Diferenciar cuándo usar un método sobre otro (ej. find vs filter).
* Solución: Análisis de los requerimientos de cada ejercicio; entendiendo que find devuelve un objeto único y filter una nueva colección.
